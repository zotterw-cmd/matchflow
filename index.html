Implementierungsstand MatchFlow v1.7.5

MatchFlow ist eine Single-Page-Web-App (eine HTML-Datei, ~11.000 Zeilen) für Fußballvereine. Firebase Auth + Firestore für Login & Datenhaltung. Multi-Tenant-fähig. Media Server via PHP (Plesk). Alle Module sind Rollen-geschützt. Stand: 22.06.2025.

Setup & Deployment
1. Dateien auf Webserver hochladen

Lade folgende Dateien per FTP in den Ordner /matchflow-api/:

  • server/upload.php → /matchflow-api/upload.php
2. Server URL eintragen

Im Feld oben links eintragen:

https://matchflow.atus-baernbach.at/matchflow-api

Dann Speichern → Verbindung testen.

3. Ordner anlegen (automatisch)

Klick auf Ordner prüfen — PHP legt automatisch an:

  • /matchflow-api/uploads/videos/
  • /matchflow-api/uploads/thumbnails/
  • /matchflow-api/uploads/players/
4. Funktionsweise
  • Spielerfotos werden unter uploads/players/ gespeichert
  • Videos & Thumbnails aus dem Skill Calculator
  • URL wird in Firestore beim Spieler gespeichert
  • Max. Upload-Größe: 50 MB
